
Machine Startup:
=================
- Power on cooler (next to compressor, the green switch)
- Power on compressor ( at the black box on the compressor the grey switch)
- Power on master switch (Large red one)

- Login at Terminal (currently BZT)
- Use Start -> Start WinCE 6 if machine does not get connected automatically
- Make sure machine is not in setup mode (Key points to 0)
- Close Toolchanger Lid
- Get zero positions
- Load a tool via M06 T1
- Spinn up via M03 S4000  (Wait about a minute)
- Spinn up to M03 S8000 (Wait about a Minute)
- Stop machine via M05

Reference Point calibration
===========================
- Mount material
- Set desired reference point (Mostly  (UAO,1)  )
- Turn on TCP
- Spin up machine M03 S5000
- Touch material at the desired position by turning the handwheel
- Set Zero coordinate 
- Repeat for all axis if required
- Move tool away from material!!
- Stop machine via M05

Milling
===========================
- Calibrate reference point
- Activate Programm
- Make sure correct tools are loaded
- On first run, reduce speed
- Start milling
- Monitor machine closely

Shutdown
===========================
- Unload Tool via M06 T0
- Switch maintanance mode on
- Clean machine (Move table via handwheel to reach all parts)
- Call Start->Shutdown on the terminal
- Switch off compressor, cooler and after the terminal is shut down, turn off main switch
